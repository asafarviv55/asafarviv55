## Featured Project

### [Enterprise RAG Engine](https://github.com/asafarviv55/enterprise-rag-engine)

A production-ready **Retrieval-Augmented Generation** system for secure document Q&A. Chat with your PDFs without sending data to public AI training sets.

```
Your Documents → Private Vector Store → Grounded AI Responses
```

**Tech Stack:**
- **LLM:** Google Gemini / OpenAI (provider-agnostic)
- **Vector DB:** ChromaDB (self-hosted)
- **API:** FastAPI with async support
- **Cache:** Redis for cost optimization
- **UI:** Streamlit

**Key Features:**
- Privacy-first architecture - data never leaves your infrastructure
- Source citations with page numbers for every response
- API key authentication & rate limiting
- Docker-ready with health checks

```bash
# Quick Start
git clone https://github.com/asafarviv55/enterprise-rag-engine.git
cd enterprise-rag-engine
docker-compose up -d
# API: http://localhost:8000/docs | UI: http://localhost:8501
```
